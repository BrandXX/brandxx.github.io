# defiantly_not_regex static subsite

This folder publishes directly to:

`https://techsoftsys.com/defiantly_not_regex/`

## Where to upload resources

- Main image: `defiantly_not_regex_b64_ocrsafe_3000.png`
- Shared page styles: `style.css`
- Rendered HTML pages: `pages/`

Source `.docx` files are intentionally kept outside the public web root:

`source-docs/defiantly_not_regex/`

## Page pattern

The main page is:

- `index.html`

Each uploaded doc should get a matching static HTML page in `pages/`, for example:

- `pages/thesis.html`
- `pages/backstory.html`
- `pages/whitepaper.html`

Then add links to those pages in the `index.html` sidebar.
